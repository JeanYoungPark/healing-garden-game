# 🌱 Healing Garden - 힐링 정원 키우기 게임 기획안

## 프로젝트 개요

**장르**: 방치형 힐링 시뮬레이션
**타겟**: 힐링을 원하는 모든 연령층
**플랫폼**: React Native (iOS/Android)
**개발 목표**: 간단하지만 퀄리티 있는 방치형 게임

---

## 🎮 게임 컨셉

### 핵심 아이디어
조용하고 평화로운 나만의 작은 정원에서 식물을 키우며 힐링하는 게임

### 차별화 포인트
- ❌ **자극적인 연출 없음**: 파티클/이펙트 최소화
- ✅ **잔잔한 힐링**: 부드러운 애니메이션, 자연 사운드
- ✅ **강제성 제거**: 스트레스 없는 자기 페이스 플레이
- ✅ **실패 없음**: 식물은 절대 죽지 않음

### 게임 루프
```
씨앗 구매 → 심기 → 물주기 → 성장 기다림 → 수확 → 골드 획득 → 새 씨앗 구매
```

---

## 🎯 핵심 메커니즘

### 1. 식물 심기
- 상점에서 씨앗 구매 (골드 사용)
- 빈 칸 터치해서 심기
- 3x3 그리드 (초기), 확장 가능

### 2. 실시간 성장 시스템
**성장 단계**:
```
씨앗 (0h) → 새싹 (1h) → 성장 (2h) → 개화 (3h) → 수확 가능
```

**특징**:
- 실제 시간 기반 (앱 종료해도 성장)
- Date 객체로 계산
- 백그라운드에서도 진행

### 3. 물주기
- 하루 최대 3번 물주기 가능
- 물 주면 성장 25% 가속 (1h → 45분)
- 물뿌리개 애니메이션
- 물 흐르는 효과음

### 4. 수확
- 개화 단계 도달 시 수확 가능
- 터치하면 골드 획득
- 반짝이는 애니메이션 (간단하게)
- 도감에 등록

### 5. 경제 시스템
```
초기 골드: 100

씨앗 가격 → 수확 골드:
- 장미: 10 → 30 (3배)
- 해바라기: 25 → 80 (3.2배)
- 튤립: 50 → 150 (3배)
- 라벤더: 100 → 320
- 벚꽃: 200 → 700
```

### Plant (식물)
```typescript
interface Plant {
  id: string;              // 고유 ID
  slotIndex: number;       // 그리드 위치 (0-8)
  type: PlantType;         // 'rose' | 'sunflower' | 'tulip' ...
```

## 💡 기술적 고려사항

### 실시간 계산
```typescript
// 성장 단계 계산
function calculateGrowth(plant: Plant): number {
  const now = new Date();
  const elapsed = now.getTime() - plant.plantedAt.getTime();
  const minutesElapsed = elapsed / 1000 / 60;

  // 물 효과 계산
  const waterBonus = plant.waterCount * 0.05;
  const adjustedMinutes = minutesElapsed * (1 + waterBonus);

  // 단계 계산 (60분/단계)
  const stage = Math.floor(adjustedMinutes / 60);
  return Math.min(stage, 3); // 최대 3단계
}
```
